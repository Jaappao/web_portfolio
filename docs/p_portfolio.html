# What's this?

これまで自分がどんなことをしてきたのかがわかるようにしたいと思ったことがきっかけで作った静的ウェブサイト。

まだHTMLやCSSの知識が不十分だったためその勉強もかねて作った。


# Target User

  • 自分
  • 自分のことを少しでも知りたいと思ってくださった方

# Development Process

ProgateやHTMLの入門書はいくつか触ったことがある程度で、自分でHTMLを書ききるほどの理解はできていなかったが、Bootstrapを使用すればなんとか形にできそうということがわかったためチャレンジしてみることにした。

画面の枠組みや配色に関して、知識が不十分で時間をかけることができなωかったので、Bootstrapや外部フレームワークに委託。それ以外のところでちょっと背伸びして頑張ってみた。ちなみにTOPページの盆栽サイズの桜っぽい(?)木の絵は自分でiPadを使って書きました。なんか挿絵が欲しいなと思って、時間があったので書いてみた。意外といい感じでかけたと自画自賛しています。拡大するとあらが出るので、各位におかれましては何卒拡大されないようお願い申し上げます。

初めてGitによるソースコード管理も行いました。複数人のグループでのGit運用は結構例に上がっているけど、自分一人での運用はどのようにやっていけばいいのかまだよくわかっていないので、ちゃんと勉強しないといけないと思いました。今回はmasterブランチにガリガリ書いている。それもよくないと思っているので、ちゃんとGitやGitHubの運用方法について知識を付けたい。
